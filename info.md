# Benni Scene Presets

Dynamic, custom light scenes for Home Assistant — works with any `light` entity,
no vendor bridge or account required.

Create your own scenes in the UI (name, image, up to 10 colours via colour picker),
and compose them into named "Looks". Designed to be driven by an external policy
that decides which scene runs when.

Based on Apache-2.0 code from `hass-scene_presets`.
